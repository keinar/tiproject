# Tipalti QA Automation Home Assignment

This repository contains the automation solution for the Tipalti Home Assignment.
It covers the **Dogs Adoption Workflow** using **Playwright** with **TypeScript**.

- **Framework:** Playwright + TypeScript
- **Design Pattern:** Page Object Model (POM)
- **Reporting:** Playwright HTML Report & GitHub Actions Integration
- **Data:** Separated test data files (Data-Driven Testing)

## üöÄ Getting Started

### Prerequisites
- Node.js (LTS version recommended)

### Installation
```bash
npm install
```

### Running the Tests
Run all tests (iterating over all dogs):
```bash
npx playwright test
```

### Viewing Reports
To view the HTML report after execution:
```bash
npx playwright show-report
```
